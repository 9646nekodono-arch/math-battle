# 計算バトル - 公開手順(Vercel)

## これは何のフォルダ?
「計算バトル」アプリを、Claudeの外(誰でも開けるWebサイト)として
公開するために必要な、プロジェクト一式です。

このフォルダをまるごとVercelにアップロードすると、公開できます。

## 一番かんたんな方法(GitHub経由)

1. https://github.com にアカウントを作る(無料)
2. 新しいリポジトリを作る(例:math-battle)
3. このフォルダの中身を、そのリポジトリにアップロードする
   (GitHubのサイト上で「Add file」→「Upload files」から、
   このフォルダの中身をまとめてドラッグ&ドロップでOK)
4. https://vercel.com にアクセスし、GitHubアカウントでログイン
5. 「Add New」→「Project」を選び、さっき作ったリポジトリを選択
6. 設定はそのまま(Framework Presetが自動でViteになるはず)で
   「Deploy」ボタンを押す
7. 数分待つと、公開用のURL(例:math-battle-xxxx.vercel.app)が発行される

## 注意点

- data(名前・コイン・進み具合)は、公開後は各端末のブラウザに保存されます。
  同じ端末・同じブラウザで開けば続きから遊べますが、
  スマホを変えたりブラウザを変えると、データは引き継がれません。
- 何かエラーが出て進めなくなったら、エラーメッセージをそのまま
  Claudeに伝えてもらえれば、一緒に解決できます。
